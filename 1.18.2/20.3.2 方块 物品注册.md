### 20.3.2 方块 / 物品注册

---

在1.18.2中，物品注册中的 `type('itemType')` 方法被移除，其注册方式变更为：

```
// 1.18.2
onEvent('item.registry', event => {
  event.create('blockid','itemType').其他方法()
})
// 1.16.5
onEvent('item.registry', event => {
  event.create('blockid').type('itemType').其他方法()
})
```

除此之外，1.18.2的方块 / 物品注册相较于1.16.5具有相同的写法，且没有任何新的发展或显著的变化。
